# 🚀 Quick Start — Desktop Shortcuts

## ✅ ¡LISTO PARA USAR!

Tenés **8 archivos .bat en tu escritorio** para acceso rápido a IDEs gratis con Ollama + Qwen3.

---

## 📋 Los 8 Shortcuts

```
Escritorio:
├── 1-Ollama-Qwen3.bat           ← Chat directo
├── 2-VS-Code-Cline.bat          ← IDE completo
├── 3-Antigravity-CrewAI.bat     ← Agentes
├── 4-Kiro-Local.bat             ← Este IDE
├── 5-Obsidian-Vault.bat         ← Notas
├── 6-Ollama-Status.bat          ← Status
├── 7-Save-Context.bat           ← Guardar
└── 8-Load-Context.bat           ← Cargar
```

## 🎯 Uso Rápido


## 💡 Tips

### Organizar en carpeta
1. Click derecho en escritorio
2. Nuevo → Carpeta
3. Nombrar: "IDEs Locales"
4. Mover los .bat ahí

### Crear tecla rápida
1. Click derecho en .bat
2. Propiedades
3. "Tecla de acceso rápido"
4. Presionar: Ctrl+Alt+Q

### Cambiar icono
1. Click derecho en .bat
2. Propiedades
3. "Cambiar icono"
4. Seleccionar uno
