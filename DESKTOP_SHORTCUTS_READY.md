## 💡 Tips

### Organizar en carpeta
1. Click derecho en escritorio
2. Nuevo → Carpeta
3. Nombrar: "IDEs Locales"
4. Mover los .bat ahí
5. Listo! Acceso organizado

### Crear tecla rápida
1. Click derecho en el .bat
2. Propiedades
3. Pestaña "Acceso directo"
4. Campo "Tecla de acceso rápido"
5. Presionar: Ctrl+Alt+Q (por ejemplo)

### Cambiar icono
1. Click derecho en el .bat
2. Propiedades
3. Cambiar icono
4. Seleccionar uno que te guste

## 🆘 Si Algo No Funciona

### "No se abre nada"
- Verificá que Ollama esté instalado
- Ejecutá: `6-Ollama-Status.bat` para ver estado

### "Ollama no inicia"
- Abrí PowerShell como admin
- Ejecutá: `ollama serve`

### "VS Code no abre"
- Verificá que VS Code esté instalado
- Ejecutá: `code --version` en PowerShell

### "CrewAI no funciona"
- Verificá que Python esté instalado
- Ejecutá: `python --version` en PowerShell
